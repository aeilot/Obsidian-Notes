1. $$

\lim_{ x \to 0 } \sin x=0

$$
2. $$
\lim_{ x \to 0 } \cos x=1
$$

3. $$

\lim_{ x \to 0 } \tan x=0

$$
$\implies$ 三角函数、反三角函数**在定义域内** $x_{0}$ 处的极限，等于函数在 $x_{0}$ 处的值
代换：负代换、$x=u+x_{0}$ ($u\to{0}$)
$\implies$ 基本初等函数**在定义域内** $x_{0}$ 处的极限，等于函数在 $x_{0}$ 处的值

4. $$
\lim_{ x \to 0 } \frac{\sin x}{x}=1
$$

注：$|\sin x|\leq|x|$ （等号仅在 $x=0$ 时成立）

5. $$

\lim_{ x \to 0 } \frac{\tan x}{x}=1

$$
6. $$
\lim_{ x \to 0 } \frac{1-\cos x}{x^2}=\frac{1}{2}
$$

7. $$

\lim_{ x \to 0 } \frac{\arcsin x}{x}=1

$$
8. $$
\lim_{ x \to 0 } \frac{\arctan x}{x}=1
$$

9. $$

\lim_{ x \to \infty } \left( 1+\frac{1}{x} \right)^x=e =\lim_{ t \to 0 } (1+t)^{\frac{1}{t}}

$$
10. $$
\lim_{ x \to 0 } \frac{\log_{a}(1+x)}{x} = \frac{1}{\ln a}
$$

特别地，$a=e$ 时，

$$
\lim_{ x \to 0 } \frac{\ln(1+x)}{x} = 1
$$

11. $$

\lim_{ x \to 0 } \frac{a^x-1}{x} = \ln a

$$
特别地，$a=e$ 时，
$$

\lim_{ x \to 0 } \frac{e^x-1}{x} = 1

$$
12. $$
\lim_{ x \to 0 } \frac{(1+x)^\alpha-1}{x}=\alpha
$$
