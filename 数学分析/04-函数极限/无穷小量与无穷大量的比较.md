# 比较

1.  若 $\lim_{x \to x_0} \frac{f(x)}{g(x)} = 0$，则记为 $f(x) = o(g(x)) \quad (x \to x_0)$。
    进而，当 $x \to x_0$ 时 $f(x), g(x)$ 均为无穷小 (大)，
    则称 $x \to x_0$ 时，$f(x)$ 是 $g(x)$ 的 **高 (低) 阶无穷小 (大)**。

2.  若 $\lim_{x \to x_0} \frac{f(x)}{g(x)} = 1 \quad (x \to x_0)$，且 $f(x), g(x)$ 同为无穷小 (大)
    则称 $x \to x_0$ 时，$f(x)$ 与 $g(x)$ 是 **等价无穷小 (大)** (可用于代换求极限 ) 
    * **特别地**：若 $g(x) = c(x-x_0)^k \quad (c \neq 0， k>0)$，
        则称当 $x \to x_0$ 时，$c(x-x_0)^k$ 为无穷小量的主部。

3.  若 $\exists M>0, \exists \mathring{U}(x_0, \delta)$，$\forall x \in \mathring{U}(x_0, \delta)$，
    都有 $|f(x)| \le M|g(x)|$，则称 $f(x) = O(g(x))$。
    * **例**: $y=x \implies y=x\sin\frac{1}{x} = O(x)$

4. 若 $\exists A, B > 0, \exists \mathring{U}(x_0)$，$\forall x \in \mathring{U}(x_0)$，有 $A \le \left|\frac{f(x)}{g(x)}\right| \le B$，且当 $x \to x_0$， 同为无穷小 (大)，则称 $f(x), g(x)$ 为 **同阶无穷小 (大)**。
	1. **特别地**: $g(x)=(x-x_0)^k$，$k$ 为无穷小 (大) 的 **阶**。

等价表示为： $f(x)\sim g(x)$ 

$f(x)=O(1)\implies$ 有界量

$f(x)=o(1)\implies$ 无穷小

注：

1. $f(x)=o(g(x))$ 有定义：$o(g(x))$ 表示函数的一个集合，即比 $g(x)$ 高阶的无穷小；
   但是 $o(g(x))=f(x)$ 没有定义
2. 等价无穷小替换：利用乘法性质 $\lim_{ x \to x_{0} }\frac{\alpha}{\beta}=\lim_{ x \to x_{0} }\frac{\alpha}{\alpha'}\frac{\alpha'}{\beta'}\frac{\beta'}{\beta}=\lim_{ x \to x_{0} }\frac{\alpha'}{\beta'}$
   代换时，对于加减法要小心 $\implies$ 加减法相当于分成几个部分，要注意极限是否存在；否则容易抵消掉主要部分
3. 常用无穷小（$x\to0$）
	1. $x\sim \sin(x)\sim \arcsin(x)\sim \tan(x)\sim \arctan(x)\sim e^x-1\sim \ln(1+x)$
	2. $1-\cos(x)\sim \frac{1}{2}x^2$
	3. $(1+x)^\alpha-1\sim \alpha x$
	4. $a^x-1\sim x\ln a$ (当 $a>0, a\neq1$)
	5. $\log_{a}(1+x)\sim \frac{x}{\ln a}$
4. $x\to{0}$ 可以替换成 $0\neq x=\phi(t)\to{0}$
   $\phi(t)\neq{0}$

### 补充：极限运算中的替换规则 

1. 加法/减法项: 一般不能用等价无穷小或等价常数替换。

    $$\lim_{x \to 0} \frac{\tan x - \sin x}{x^3}$$

    错误: $\lim_{x \to 0} \frac{x - x}{x^3} = 0$.

    

    正确:

    $$\lim_{x \to 0} \frac{\tan x - \sin x}{x^3} = \lim_{x \to 0} \frac{\sin x (\frac{1}{\cos x} - 1)}{x^3} = \lim_{x \to 0} \frac{x (1 - \cos x)}{x^3 \cos x} = \lim_{x \to 0} \frac{x \cdot \frac{1}{2} x^2}{x^3 \cdot 1} = \frac{1}{2}$$

2. $e^{x^2} - \cos x$

    $$\lim_{x \to 0} \frac{e^{x^2} - \cos x}{x^2}$$

    错误: $\lim_{x \to 0} \frac{(1 + x^2) - (1 - \frac{1}{2} x^2)}{x^2} = \lim_{x \to 0} \frac{\frac{3}{2} x^2}{x^2} = \frac{3}{2}$ (此处原笔记划去了用等价无穷小替换的错误尝试，并给出了泰勒展开的正确思路)。

    利用泰勒展开: $e^u = 1 + u + o(u)$, $\cos u = 1 - \frac{u^2}{2!} + o(u^2)$。

    $$\lim_{x \to 0} \frac{e^{x^2} - \cos x}{x^2} = \lim_{x \to 0} \frac{(1 + x^2 + o(x^2)) - (1 - \frac{x^2}{2} + o(x^2))}{x^2}$$

    $$= \lim_{x \to 0} \frac{x^2 + \frac{x^2}{2} + o(x^2)}{x^2} = \lim_{x \to 0} \left(\frac{3}{2} + \frac{o(x^2)}{x^2}\right) = \frac{3}{2}$$

3. 乘法项/除法项: 一般不能用等价无穷小或等价常数替换。

    $$\lim_{x \to 0} \frac{x - \sin x \cos x}{x^3}$$

    错误: $\lim_{x \to 0} \frac{x - x \cdot 1}{x^3} = 0$.

    正确:

    $$\lim_{x \to 0} \frac{x - \sin x \cos x}{x^3} = \lim_{x \to 0} \frac{x - \frac{1}{2} \sin 2x}{x^3} = \lim_{x \to 0} \frac{x - \frac{1}{2} (2x - \frac{(2x)^3}{3!} + o(x^3))}{x^3}$$
    $$= \lim_{x \to 0} \frac{x - x + \frac{4}{6} x^3 + o(x^3)}{x^3} = \lim_{x \to 0} \frac{\frac{2}{3} x^3}{x^3} = \frac{2}{3}$$
