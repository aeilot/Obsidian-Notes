### 1. 核心定义

* **是什么**: 一种用 `(r, θ)`（距离, 角度）来定位点的二维坐标系统。
* **极点 (Pole)**: 坐标系的原点 (0, 0)。
* **极轴 (Polar Axis)**: 从极点出发的水平射线，相当于直角坐标的 x 轴正方向。

### 2. 两个关键元素

* **极径 `r` (Radius)**: 点到极点的**距离**。$r$ 可以为负，表示沿相反方向延伸。
* **极角 `θ` (Theta/Angle)**: 从极轴**逆时针**旋转到目标点连线所成的**角度**。

### 3. 与直角坐标 $(x, y)$ 的转换

#### 从极坐标到直角坐标

* $x = r \cos(\theta)$
* $y = r \sin(\theta)$

#### 从直角坐标到极坐标

* $r = \sqrt{x^2 + y^2}$
* $\theta = \arctan(\frac{y}{x})$ (注意根据点所在象限调整角度)

### 4. 常见极坐标方程与图形

| 图形种类          | 标准方程                      | 特征描述                                         |
| :---------------- | :---------------------------- | :----------------------------------------------- |
| **圆 (Circle)** | `$r = a$`                     | 圆心在极点，半径为 a。                           |
|                   | `$r = 2a \cos(\theta)$`        | 圆心在 $(a, 0)$，半径为 a，过极点。              |
| **心形线 (Cardioid)** | `$r = a(1 \pm \cos(\theta))$`  | 形状像爱心，开口方向由正负号和 `cos`/`sin` 决定。 |
| **玫瑰线 (Rose Curve)** | `$r = a \cos(n\theta)$`        | n 为奇数，有 n 个花瓣。<br>n 为偶数，有 2n 个花瓣。 |
| **螺线 (Spiral)** | `$r = a\theta$`               | 从极点向外盘旋的曲线。                           |
| **双纽线 (Lemniscate)**| `$r^2 = a^2 \cos(2\theta)$`    | 形状像横放的 "8" 或无穷符号 $\infty$。            |
