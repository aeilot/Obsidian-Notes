# 维数与基

在线性空间 $V$ 中，如果存在 $n$ 个元素 $\mathbf{\alpha}_1, \mathbf{\alpha}_2, \dots, \mathbf{\alpha}_n$ 满足：

1. $\mathbf{\alpha}_1, \mathbf{\alpha}_2, \dots, \mathbf{\alpha}_n$ 线性无关；
2. $V$ 中任一元素 $\mathbf{\alpha}$ 总可由 $\mathbf{\alpha}_1, \mathbf{\alpha}_2, \dots, \mathbf{\alpha}_n$ 线性表示，

那么 $\mathbf{\alpha}_1, \mathbf{\alpha}_2, \dots, \mathbf{\alpha}_n$ 称为线性空间 $V$ 的一个**基** (Basis)，$n$ 称为线性空间的**维数** (Dimension)，记 $\dim V = n$

维数为 $n$ 的线性空间称为 $n$ 维线性空间，记作 $V_n$

## 四个基本子空间

列空间：$\{ Ax \mid  \forall  x \in R^n\}$，$dimC(A)=n$

行空间：$\{ x^TA \mid  \forall  x \in R^m\}$，$dimC(A^T)=m$

（齐次线性方程组的）解空间：$\{ x \mid Ax=0 \}$，也称为矩阵 $A$ 的零空间，$dimN(A)=n-r$

左零空间：$\{ x\mid A^Tx=0 \}$ ，$dim(N(A^T))=m-r$

## 维数公式

设 $V$ 是有限维线性空间，$U_1, U_2$ 均是 $V$ 的子空间，则有：

$$\dim(U_1 + U_2) = \dim(U_1) + \dim(U_2) - \dim(U_1 \cap U_2)$$

> [!NOTE] 证明思路详解
> 
> #### 步骤 1：从“交集”开始
> 
> 1. $U_1 \cap U_2$ 本身也是一个子空间。
>     
> 2. 设 $\dim(U_1 \cap U_2) = k$。
>     
> 3. 我们取 $U_1 \cap U_2$ 的一个基：
>
>     $$\mathbf{B}_{1 \cap 2} = \{\mathbf{\gamma}_1, \dots, \mathbf{\gamma}_k\}$$
>
>     这 $k$ 个向量是线性无关的，并且它们既属于 $U_1$ 也属于 $U_2$。
>     
> 
> #### 步骤 2：扩充基到 $U_1$ 和 $U_2$
> 
> 4. **对于 $U_1$**：
>     
>     - $\mathbf{B}_{1 \cap 2}$ 是 $U_1$ 中 $k$ 个线性无关的向量。
>         
>     - 根据**基扩充定理**，我们可以添加 $m-k$ 个新向量 $\{\mathbf{\alpha}_1, \dots, \mathbf{\alpha}_{m-k}\}$，使它成为 $U_1$ 的一个基。
>         
>     - $U_1$ 的基 $\mathbf{B}_1 = \{\mathbf{\gamma}_1, \dots, \mathbf{\gamma}_k, \mathbf{\alpha}_1, \dots, \mathbf{\alpha}_{m-k}\}$
>         
>     - 因此，$\dim(U_1) = k + (m-k) = m$。
>         
> 5. **对于 $U_2$**：
>     
>     - 同理，我们也可以扩充 $\mathbf{B}_{1 \cap 2}$，添加 $n-k$ 个新向量 $\{\mathbf{\beta}_1, \dots, \mathbf{\beta}_{n-k}\}$，使它成为 $U_2$ 的一个基。
>         
>     - $U_2$ 的基 $\mathbf{B}_2 = \{\mathbf{\gamma}_1, \dots, \mathbf{\gamma}_k, \mathbf{\beta}_1, \dots, \mathbf{\beta}_{n-k}\}$
>         
>     - 因此，$\dim(U_2) = k + (n-k) = n$。
>         
> 
> #### 步骤 3：构造 $U_1 + U_2$ 的基（最关键一步）
> 
> 我们现在声称，把上面三组向量**合并**在一起，就构成了 $U_1 + U_2$ 的一个基：
>
> $$\mathbf{B}_{1+2} = \{\underbrace{\mathbf{\gamma}_1, \dots, \mathbf{\gamma}_k}_{\text{来自交集}}, \underbrace{\mathbf{\alpha}_1, \dots, \mathbf{\alpha}_{m-k}}_{\text{U}_1 \text{独有}}, \underbrace{\mathbf{\beta}_1, \dots, \mathbf{\beta}_{n-k}}_{\text{U}_2 \text{独有}}\}$$
>
> 要证明 $\mathbf{B}_{1+2}$ 是 $U_1 + U_2$ 的基，我们必须（根据你第一张图的定义）证明它满足两点：
> 
> 6. **生成性（Spanning）**：
>     
>     - $U_1 + U_2$ 中任意的向量 $\mathbf{v}$ 都可以写成 $\mathbf{v} = \mathbf{u}_1 + \mathbf{u}_2$，其中 $\mathbf{u}_1 \in U_1, \mathbf{u}_2 \in U_2$。
>         
>     - $\mathbf{u}_1$ 可以由 $\mathbf{B}_1$（$\mathbf{\gamma}$ 和 $\mathbf{\alpha}$ 组）线性表示。
>         
>     - $\mathbf{u}_2$ 可以由 $\mathbf{B}_2$（$\mathbf{\gamma}$ 和 $\mathbf{\beta}$ 组）线性表示。
>         
>     - 因此，它们的和 $\mathbf{v}$ 必定可以由 $\mathbf{B}_{1+2}$（$\mathbf{\gamma}, \mathbf{\alpha}, \mathbf{\beta}$ 三组）线性表示。
>         
>     - 所以，$\mathbf{B}_{1+2}$ **能生成** $U_1 + U_2$。
>         
> 7. **线性无关（Linear Independence）**：
>     
>     - 这是证明的核心难点。我们需要证明，如果：
>
>         $$\sum c_i \mathbf{\gamma}_i + \sum a_j \mathbf{\alpha}_j + \sum b_l \mathbf{\beta}_l = \mathbf{0}$$
>
>     - 那么唯一的解是所有的系数 $c_i, a_j, b_l$ 全都为 0。
>         
>     - 证明技巧：把 $\mathbf{\beta}$ 组移到等式右边：
>
>         $$\underbrace{\sum c_i \mathbf{\gamma}_i + \sum a_j \mathbf{\alpha}_j}_{\text{向量 } \mathbf{x}} = \underbrace{-\sum b_l \mathbf{\beta}_l}_{\text{向量 } \mathbf{y}}$$
>
>     - 看左边：向量 $\mathbf{x}$ 是 $U_1$ 基向量的线性组合，所以 $\mathbf{x} \in U_1$。
>         
>     - 看右边：向量 $\mathbf{y}$ 是 $U_2$ 基向量的线性组合，所以 $\mathbf{y} \in U_2$。
>         
>     - 因为 $\mathbf{x} = \mathbf{y}$，所以这个向量**既在 $U_1$ 中，又在 $U_2$ 中**，这意味着 $\mathbf{x} \in (U_1 \cap U_2)$。
>         
>     - 既然 $\mathbf{x} \in (U_1 \cap U_2)$，它就必须能被 $U_1 \cap U_2$ 的基（$\mathbf{\gamma}$ 组）线性表示。
>
>         $$\mathbf{x} = \sum d_i \mathbf{\gamma}_i$$
>
>     - 所以我们有：
>
>         $$\sum d_i \mathbf{\gamma}_i = \mathbf{x} = -\sum b_l \mathbf{\beta}_l \implies \sum d_i \mathbf{\gamma}_i + \sum b_l \mathbf{\beta}_l = \mathbf{0}$$
>
>     - 但是，$\{\mathbf{\gamma}_i\}$ 和 $\{\mathbf{\beta}_l\}$ 共同组成了 $U_2$ 的基 $\mathbf{B}_2$，它们是**线性无关**的！
>         
>     - 因此，唯一的可能是所有系数都为 0，即 $d_i = 0$ 且 $\mathbf{b_l = 0}$。
>         
>     - 我们成功证明了 $\mathbf{b_l = 0}$。
>         
>     - 把 $b_l = 0$ 代回最初的等式：
>
>         $$\sum c_i \mathbf{\gamma}_i + \sum a_j \mathbf{\alpha}_j = \mathbf{0}$$
>
>     - 而 $\{\mathbf{\gamma}_i\}$ 和 $\{\mathbf{\alpha}_j\}$ 共同组成了 $U_1$ 的基 $\mathbf{B}_1$，它们也是**线性无关**的！
>         
>     - 因此，$\mathbf{c_i = 0}$ 且 $\mathbf{a_j = 0}$。
>         
>     - 至此，我们证明了所有系数 $a_j, b_l, c_i$ 必须全为 0。$\mathbf{B}_{1+2}$ **是线性无关的**。
>         
> 
> #### 步骤 4：计数与总结
> 
> 8. 我们已经证明 $\mathbf{B}_{1+2}$ 是 $U_1 + U_2$ 的一个基。
>     
> 9. 那么 $\dim(U_1 + U_2)$ 就等于 $\mathbf{B}_{1+2}$ 中向量的个数。
>     
> 10. $\dim(U_1 + U_2) = (\text{k 个 } \mathbf{\gamma}) + (\text{(m-k) 个 } \mathbf{\alpha}) + (\text{(n-k) 个 } \mathbf{\beta})$
>     
> 11. $\dim(U_1 + U_2) = k + (m-k) + (n-k) = m + n - k$
>     
> 12. 把 $m, n, k$ 换回它们所代表的维数：
>     
>     - $m = \dim(U_1)$
>         
>     - $n = \dim(U_2)$
>         
>     - $k = \dim(U_1 \cap U_2)$
>         
> 13. 代入得到：
>
>     $$\dim(U_1 + U_2) = \dim(U_1) + \dim(U_2) - \dim(U_1 \cap U_2)$$
>
> 证明完毕。
> 
### 推论 (Coro)

设 $V$ 是有限维线性空间，$U_1, U_2$ 均是 $V$ 的子空间，如果 $V = U_1 \oplus U_2$（直和），则有：

$$\dim(V) = \dim(U_1 + U_2) = \dim(U_1) + \dim(U_2)$$
