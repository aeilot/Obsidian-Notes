# 向量组

- **向量组的线性表示**
    - 设 $\Sigma_1$ 与 $\Sigma_2$ 为 $K^n$ 中的两个向量组。
    - 如果 $\Sigma_2$ 中的每一个向量都能表示为 $\Sigma_1$ 中有限个向量的线性组合，则称向量组 $\Sigma_2$ 能由向量组 $\Sigma_1$ 线性表示。
- **等价向量组**
    - 如果向量组 $\Sigma_1$ 与 $\Sigma_2$ 能**互相**线性表示，则称这两个向量组是**等价 (equivalent)** 的。
    - 记作：$\Sigma_1 \sim \Sigma_2$。
