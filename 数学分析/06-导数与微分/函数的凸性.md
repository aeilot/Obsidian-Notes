# 函数的凸性与拐点

- Def 4.2 (凸函数): 设函数 $f(x)$ 定义在区间 $I$ 上，若对 $\forall x_1, x_2 \in I$ 和 $\forall \lambda \in (0, 1)$，总有：

$$f(\lambda x_1 + (1-\lambda)x_2) \le \lambda f(x_1) + (1-\lambda)f(x_2)$$

则称 $f(x)$ 为 $I$ 上的下凸函数 (习惯上称为凸函数)。

- **注 1:** 若 “$\le$” 变为 “$<$”，称 $f(x)$ 为 $I$ 上的严格凸函数。
        
    - **注 2:** 不等号反向，则为上凸函数。
        
**Thm 4.13 (凸函数的等价命题):** 设 $f(x)$ 定义在区间 $I$ 上，则以下命题等价：
    
- I) $f(x)$ 为 $I$ 上的凸函数。
        
- II) 对 $\forall x_1, x_2, x_3 \in I$ ($x_1 < x_2 < x_3$)，有 (割线斜率递增)：

        $$\frac{f(x_2) - f(x_1)}{x_2 - x_1} \le \frac{f(x_3) - f(x_2)}{x_3 - x_2}$$

- III) 若 $f(x)$ 在 $I$ 上可导，对 $\forall x_1, x_2 \in I$，有 (函数在切线上方)：

$$f(x_2) \ge f(x_1) + f'(x_1)(x_2 - x_1)$$

- **注:** II)、III) 均有几何意义。
        
- **Thm 4.14 (凸性的第一判别法):** 设 $f \in C[a, b] \cap D(a, b)$，则 $f(x)$ 在 $[a, b]$ 上为凸函数的充要条件是 $f'(x)$ 在 $(a, b)$ 内递增。
    
- **Thm 4.15 (凸性的第二判别法):** 设 $f \in C[a, b] \cap D^2(a, b)$，则 $f(x)$ 在 $[a, b]$ 上为凸函数的充要条件是：对 $\forall x \in (a, b)$，有 $f''(x) \ge 0$。
    
- **Def 4.3 (拐点):** 设函数 $f(x)$ 在 $(a, b)$ 内连续。若 $x_0 \in (a, b)$ 为 $f(x)$ 的上凸与下凸的分界点，则 $x_0$ 为 $f(x)$ 的拐点；相应地，点 $(x_0, f(x_0))$ 称为曲线 $y=f(x)$ 的拐点。
    
    - **注:** 函数的拐点可能在 $f''(x_0) = 0$ 或 $f''(x_0)$ 不存在的点。

$$
2. I 区间上 $x_{0}$ 为内点，则 $f'_{-}(x_{0})$、$f'_{+}(x_{0})$ 存在，且  $f'_{-}(x_{0})\leq f'_{+}(x_{0})$ 
3. 凸函数在内点连续（端点不一定连续）
# Jensen 不等式

如果 $f$ 是一个凸函数， $x_1, x_2, \dots, x_n$ 是定义域中的点， $\lambda_1, \lambda_2, \dots, \lambda_n$ 是非负权重，且 $\lambda_1 + \lambda_2 + \dots + \lambda_n = 1$，则：

$$f(\sum_{i=1}^n \lambda_i x_i) \le \sum_{i=1}^n \lambda_i f(x_i)$$
