# 高阶导数

二阶导数 $f''(x)$、$\frac{d^2y}{dx^2}$

三阶导数 $f'''(x)$、$\frac{d^3y}{dx^3}$

四阶导数 $f^{(4)}(x)$、$\frac{d^4y}{dx^4}$

n 阶导数 $f^{(n)}(x)$、$\frac{d^ny}{dx^n}$

$\implies$ $f^n(x)$ 表示 $n$ 次复合函数

>[!NOTE] 注意
>1. $f^{(n)}(x)$ $n\geq 2$ 时，称为高阶导数
>2. $f^{(n)}(x_{0})$ 存在 $\implies$ $f^{(n-1)}(x)$ 在 $U(x_{0})$ 内有定义，且在 $x_{0}$ 处连续

## 例子

- 正弦函数 $(\sin x)^{(n)} = \sin(x + \frac{n\pi}{2})$
    
- 余弦函数 $(\cos x)^{(n)} = \cos(x + \frac{n\pi}{2})$
    
- 对数函数 $(\ln(1+x))^{(n)} = \frac{(-1)^{n-1}(n-1)!}{(1+x)^n}$
    
- 分式函数 $(\frac{1}{ax+b})^{(n)} = \frac{(-1)^n n! a^n}{(ax+b)^{n+1}}$
