
### L1 距离 (Manhattan Distance | 曼哈顿距离) 

L1 距离是两个向量对应维度差值的**绝对值之和**。 其计算公式为： $$L1(X, Y) = \sum_{i=1}^{n} |x_i - y_i| = |x_1 - y_1| + |x_2 - y_2| + ... + |x_n - y_n|$$

### L2 距离 (Euclidean Distance | 欧几里得距离) 

L2 距离是两个向量对应维度差值的**平方和，再开平方根**，也就是我们通常所说的两点之间的直线距离。 其计算公式为： $$L2(X, Y) = \sqrt{\sum_{i=1}^{n} (x_i - y_i)^2} = \sqrt{(x_1 - y_1)^2 + (x_2 - y_2)^2 + ... + (x_n - y_n)^2}$$

| 特性 (Feature) | L1 (曼哈顿距离)                                        | L2 (欧几里得距离)                                                    |
| :----------- | :------------------------------------------------ | :------------------------------------------------------------- |
| **几何形状**     | 与原点等距的点构成**正方形**或**正多面体** (有尖角)。                  | 与原点等距的点构成**圆形**或**球面** (光滑)。                                   |
| **对异常值的敏感度** | **较低 (更鲁棒)**。<br>差值不经过平方，避免了对单个大误差的过度放大。          | **较高 (更敏感)**。<br>对差值进行平方，放大了异常值的影响。                            |
| **主要应用**     | 1.  **特征选择**<br>2.  当数据中存在很多无用特征时<br>3.  需要鲁棒的模型时 | 1.  **防止过拟合** (权重衰减)<br>2.  当所有特征都被认为有用时<br>3.  作为大多数算法的默认距离度量 |

L2: 平方放大数值，放大了异常值影响
