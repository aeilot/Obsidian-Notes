利用康托 - 施罗德 - 伯恩斯坦定理 (Cantor-Schröder-Bernstein Theorem)

该定理指出，如果存在从集合 A 到集合 B 的一个单射（一对一注入），并且存在从集合 B 到集合 A 的一个单射，那么 A 和 B 的基数相等（即存在一个从 A 到 B 的双射）。

为了证明 $|\mathbb{I}| = |\mathbb{R}|$，我们需要完成以下两步：

1.  **证明 $|\mathbb{I}| \le |\mathbb{R}|$**:
    * 我们需要构造一个从 $\mathbb{I}$ 到 $\mathbb{R}$ 的单射函数。
    * 这非常简单，因为 $\mathbb{I}$ 本身就是 $\mathbb{R}$ 的一个子集。我们可以定义一个包含映射（inclusion map）$f: \mathbb{I} \to \mathbb{R}$，其中对任意 $x \in \mathbb{I}$，都有 $f(x) = x$。
    * 这个函数显然是单射。因此，我们证明了 $|\mathbb{I}| \le |\mathbb{R}|$。

2.  **证明 $|\mathbb{R}| \le |\mathbb{I}|$**:
    * 我们需要构造一个从 $\mathbb{R}$ 到 $\mathbb{I}$ 的单射函数。这是证明中比较巧妙的部分。
    * 我们可以利用小数展开来构造这个函数。考虑一个实数 $x \in \mathbb{R}$，它有一个标准十进制展开（为保证唯一性，我们不使用以无限个 9 结尾的表示，例如用 0.5 代替 0.4999...）。
    * $x = \pm n.d_1 d_2 d_3 d_4 \dots$
    * 我们可以定义一个函数 $g: \mathbb{R} \to \mathbb{I}$，它将 $x$ 的小数部分的每一位数字 $d_i$ 映射到一个非零数字，然后将这些数字与一个保证无理性的结构交织在一起。
    * 一个简单的构造如下：定义一个映射 $h$ 将数字 $\{0, 1, 2, ..., 9\}$ 映射到 $\{1, 2, ..., 10\}$，例如 $h(d) = d+1$。
    * 现在定义函数 $g(x)$ 如下：
        * 取 $x$ 的小数展开 $d_1 d_2 d_3 \dots$。
        * 构造一个新的小数 $y = 0.h(d_1)1h(d_2)11h(d_3)111h(d_4)\dots$
        * 即我们在每个转换后的数字 $h(d_i)$ 后面插入一个由 $i$ 个 "1" 组成的块。
    * 例如，如果 $x = 0.305...$，则 $h(3)=4, h(0)=1, h(5)=6$。那么 $g(x) = 0.411116111...$。
    * **这个函数是单射的**：如果两个不同的实数 $x_1$ 和 $x_2$ 有不同的小数展开，那么它们构造出的新小数 $g(x_1)$ 和 $g(x_2)$ 显然也不同。
    * **这个函数的输出总是无理数**：构造出的数 $g(x)$ 的小数部分不可能是循环的，因为我们插入的 "1" 的块的长度（1, 2, 3, ...）一直在增加，破坏了任何可能的循环节。同时，因为它包含无限个非零数字，所以它也不是有穷小数。因此，$g(x)$ 永远是无理数。
    * 通过构造这个从 $\mathbb{R}$ 到 $\mathbb{I}$ 的单射函数 $g$，我们证明了 $|\mathbb{R}| \le |\mathbb{I}|$。

**最终结论**:

我们已经证明了 $|\mathbb{I}| \le |\mathbb{R}|$ 和 $|\mathbb{R}| \le |\mathbb{I}|$。根据康托 - 施罗德 - 伯恩斯坦定理，我们可以得出结论：

$$|\mathbb{I}| = |\mathbb{R}| = c$$

因此，无理数集的势等于实数集的势，即连续统势。这直观上意味着无理数的“数量”和实数的“数量”是一样多的。
