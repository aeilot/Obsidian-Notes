- 关键定理：对于任意实数矩阵 $A$，其零空间 $N(A)$ 与 $N(A^T A)$ 是相同的。即：

    $$N(A) = N(A^T A)$$

- **证明速览**：
    
    1. 若 $x \in N(A)$，则 $Ax=0$。两边左乘 $A^T$ 得 $A^T A x = 0$，所以 $x \in N(A^T A)$。
        
    2. 若 $x \in N(A^T A)$，则 $A^T A x = 0$。两边左乘 $x^T$ 得 $x^T A^T A x = 0$，即 $(Ax)^T (Ax) = 0$。
        
    3. 令 $y = Ax$，则 $y^T y = 0$。$y^T y$ 是 $y$ 中所有元素平方和，它为 0 当且仅当 $y$ 为零向量。
        
    4. 因此 $y = Ax = 0$，这意味着 $x \in N(A)$。
        
- **结论**：$A^T A x = 0$ 的解与 $Ax = 0$ 的解是**完全相同**的。
# 17. Sherman-Morrison 公式（用于求逆）

这个公式给出了 $(A + \mathbf{u}\mathbf{v}^T)$ 的逆。

**定理内容：**

设 $A$ 是一个 $n$ 阶可逆矩阵，$\mathbf{u}$ 和 $\mathbf{v}$ 是 $n$ 维列向量。

如果 $A + \mathbf{u}\mathbf{v}^T$ 可逆，那么其逆矩阵为：

$$(A + \mathbf{u}\mathbf{v}^T)^{-1} = A^{-1} - \frac{A^{-1}\mathbf{u}\mathbf{v}^T A^{-1}}{1 + \mathbf{v}^T A^{-1}\mathbf{u}}$$

可逆的充分必要条件：

$A + \mathbf{u}\mathbf{v}^T$ 可逆的充分必要条件是 $1 + \mathbf{v}^T A^{-1}\mathbf{u} \neq 0$。

> **注意：** $1 + \mathbf{v}^T A^{-1}\mathbf{u}$ 是一个标量（$1 \times 1$ 矩阵），因为它是一个 (1×n) × (n×n) × (n×1) 运算的结果。
