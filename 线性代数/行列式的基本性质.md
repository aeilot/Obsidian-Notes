# 行列式的转置

$$
D = |A|,\;D^T = |A^T|
$$

$D^T$ 称为 $D$ 的**转置**或**转置行列式**，也记为 $D'$

$$
\det(A) = \det(A^T)
$$

$\implies$ 行列式中，行与列对称

## 性质
1. 对换行列式的两行，行列式变号
2. 若行列式中有两行相同，行列式的值为 0
3. 若行列式中有两行对应元素成比例，行列式的值为 0
4. 将行列式中某一行的常数倍加到另一行，行列式的值不变
